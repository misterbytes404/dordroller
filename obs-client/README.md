# OBS Client Setup Instructions

## How to Use in OBS

1. Start the backend server
2. Open this `index.html` file in a browser or serve it locally
3. In OBS, add a **Browser Source**
4. Set the URL to: `file:///path/to/obs-client/index.html?room=YOUR_ROOM_CODE`
5. Set dimensions (recommended: 1920x1080)
6. Check "Shutdown source when not visible" for performance
7. Adjust position/scale as needed for your stream layout

## URL Parameters

- `room` - Set the room code to listen to (e.g., `?room=game1`)

## Customization

Edit `styles.css` to customize:
- Colors and gradients
- Font sizes
- Animation timing
- Display duration
